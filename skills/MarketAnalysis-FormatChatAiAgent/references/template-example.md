## Key Rules (from this example)

- **Bold `** **`** for: main title, all section titles
- **Code block ` ``` `** for: technical table, history data ONLY
- **Plain text** for: Fundamental paragraph, Price Movement, Waiting Sell/Buy Area, Mirroring Assistant
- **Links MUST be embedded** in Fundamental using `[text](link)` — EVERY news source needs a link
- **Fear & Greed MUST** be in Fundamental with link to alternative.me
- RSI + Volume outside code block, plain text
- History Data: current day only, MUST appear even if just Waiting Order
- Analyst number increments: Analyst1, Analyst2, Analyst3
- Astronacci wave analysis in Price Movement — name the wave count
